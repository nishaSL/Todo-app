# Simple Todo App 


##What are the features?
*Sign in
*Sign up
*Sessions
*Validation
*Authentication
*Adding todos
*Removing todos

##The ToDo app uses the following technologies and javascript libraries:

*Express.js
*Node.js
*express-ejs
*express-session
*SQLite3

##Installing
You need to have installed Node.js, NPM and SQlite3 in your System.
